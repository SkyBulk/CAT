<h2><a aria-hidden="true" class="anchor" href="#Ethernet/IP" id="user-content-Ethernet/IP"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Ethernet/IP</h2>
<p>Use of the industrial Ethernet / IP protocol to obtain information via port 44818 TCP and UDP.</p>
<p><strong>enip-info.nse</strong> Script developed in lua for nmap. This script sends a request to port 44818 TCP used by the Ethernet / IP protocol and parses information such as device type, manufacturer identifier, product name, serial number, device status, etc.</p>

<p>Use of the industrial modbus TCP protocol to obtain information through the 502 TCP port.</p>
<p><strong>modbus-discover.nse</strong> Script developed in lua for nmap. This script lists the IDs of the slaves (sids) and collects device information.</p>

<p>Use of the FINS protocol to obtain PLC information in industrial networks. This protocol works both under UDP and TCP on port 9600.</p>
<p><strong>omron-info.nse</strong> Script developed in lua for nmap. This script sends a FINS package to the device that contains a control command for parameter reading. If the command is valid, the device responds with information that is parsed by the script and displayed on the screen.</p>

<p>Use of the MMS industrial protocol for obtaining information through port 102 TCP.</p>
<p><strong>mms-identify.nse</strong> Script developed in lua for nmap. This script verifies that the IEC 61850-8-1 protocol exists through Port 102 TCP to subsequently send a request in order to extract information from the device.</p>
